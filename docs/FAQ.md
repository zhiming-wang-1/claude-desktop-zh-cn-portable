# FAQ

## 这个仓库会不会修改官方 Claude Desktop？

设计目标是不修改官方 Claude Desktop，而是生成一个独立中文绿色版副本。官方版建议保持原样。

## 为什么不能直接上传已经生成好的绿色版？

因为生成后的绿色版包含 Claude 官方程序本体。本仓库只能开源工具、词库和文档，不能分发 Claude 官方二进制文件或安装包。

## 登录后网页回调打开了官方版怎么办？

运行工具菜单，选择：

```text
12 Dual launch / OAuth login repair
```

它会临时把 `claude://` 登录回调指向中文绿色版启动器。登录完成后，可以按工具提示恢复或继续保持绿色版优先。

## 为什么不要直接双击绿色版里的 Claude.exe？

直接双击可能不会带上独立用户数据目录，也可能绕过兼容启动器。推荐始终用桌面或开始菜单里的 `Claude zh-CN` 快捷方式。

## 卸载官方 Claude 会影响绿色版吗？

已经生成好的绿色版通常可以继续运行。但以后更新或重建绿色版时，仍需要本机官方版或官方来源 MSIX。稳妥做法是保留官方版作为原版备用。

## 直接汉化官方版和绿色版有什么区别？

直接汉化官方版会改官方安装目录，尤其 Microsoft Store/Appx 版更容易出现重启后打不开、更新失败或签名校验问题。绿色版是修改独立副本，更适合普通用户。
