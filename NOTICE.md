# Notice

This repository is a community-maintained localization and portable-copy helper for Claude Desktop on Windows.

It contains scripts, translation resources, documentation, and compatibility helpers only. It does not contain Claude Desktop binaries, official installers, MSIX/Appx packages, account data, API keys, or access tokens.

The project is based on the MIT-licensed community work around `CC-desktop-zh-cn-WIN-portable` and keeps the same general safety boundary: generate or update a local portable copy from software the user already has installed locally or downloads from official sources.

Claude, Claude Desktop, Claude Code, Anthropic, and related trademarks belong to their respective owners. This project is not affiliated with, endorsed by, sponsored by, or officially connected to Anthropic.

When publishing forks or releases, do not include:

```text
Claude.exe
app.asar
icudtl.dat
official MSIX/Appx installers
unpacked Claude application directories
account data
cookies
tokens
API keys
generated user data
```
